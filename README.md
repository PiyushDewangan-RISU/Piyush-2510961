## Memory Card Flip-Match Game

## Project Name: Memory Card Game

## Description

A simple flip–match memory game where players flip cards to find matching pairs.

Built using HTML, CSS, and JavaScript, the game tracks moves and matched pairs.

## Features

• Flip cards to reveal symbols

• Match pairs of cards to score points

• Tracks moves and matched pairs

• Restart game anytime

• Responsive design for desktop and mobile

## Technologies Used

• HTML5 – Webpage structure

• CSS3 – Styling, grid layout, transitions, and hover effects

• JavaScript – DOM manipulation, event handling, game logic

## How to Play

• Click a card to flip it.

• Click a second card to try and match it with the first one.

• If the cards match, they remain flipped; otherwise, they flip back.

• Continue until all pairs are matched.

• Track your moves and matched pairs to see your performance.
